### 转专业
#### 课程学分转换
1. **专业课转校选**
   + 建议大一第二学期修一门美育课，其它校选课可以用专业课
   + 和信院计算机相关的课无法转换成校选
   + 电院大一年级平转信院，校选课转换分很少（可能只有机械制图），需要另外修校选
   + <mark>转专业前选了信息学院开设的校选课，学分是无效的</mark>
2. **数学课**
   + 转专业前修过《微积分Ⅰ》，为同一课程
   + 《线性代数Ⅰ》学分可以转成《线性代数A》免修，但是原绩点不计算。
3. **物理课**
   + 《大学物理A》上下都修完了，可以合并一起转换成《大学物理B》上下。不能上单独转换，也不能下单独转换。
   + 物理学院另说
4. **英语课**
   + 英语专业另说
5. **电路原理**
   + 电院《电路分析》学分可以转成《电路原理》免修，但是原绩点不计算。
   + 修了航院等学院的《电路原理》，也不用修信院《电路原理》

### 美食
> 🦐民以食为天

#### 翔安校区食堂
1. **竞丰餐厅（翔安一期食堂）**
绿豆汤好喝，清凉解暑。
感觉一楼比较适合我这种~~长期素食~~爱吃菜的杂食主义者，但是也褒贬不一。黄焖鸡米饭还行。
开封菜，无需多言！不过每周四都会爆满，有课的话慎点。

2. **丰庭餐厅（翔安二期食堂）**
绿豆汤好喝，清凉解暑。
一楼小卖部的粥和炒粉，很适合下午七八节学武楼的课无缝衔接。
一楼的果汁挺可口的，就是说原来橙汁和芒果汁容易分不清。
🦐一楼卖的广式茶点虾饺不错。
偶尔点个一楼的不麻不辣不烫的麻辣烫，自己搭配喜欢食材，来调剂胃口。
这个地理位置，这个大上坡，对自行车党很不友好（）

3. **思源餐厅（翔安三期食堂）**
绿豆汤好喝，清凉解暑。
一楼每年都会有一次免费豆浆，是用同学的作业大豆磨的。顺便一提，每个食堂的豆浆都挺好喝的。
三楼很贵，但是可以多吃几种种类。小心菜品刺客（虾仁滑蛋、卤牛肉，非常好吃，但是很贵！）。一定要记得抢免费蛋糕和水果吃回本

4. **芙蓉餐厅（翔安四期食堂）**
绿豆汤好喝，清凉解暑。
贵，但是口味好。有时甚至有蒜蓉扇贝、大闸蟹……
铁板面很香。

> [!TIP]
> 如果懒得排队的话，可以用厦园食光小程序，优点是比较快，而且不是国潮外卖，缺点是完全没有新鲜感，多花一点配送费
