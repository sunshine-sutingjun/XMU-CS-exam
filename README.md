## 🤝如何贡献

我们欢迎所有同学的贡献！请按照以下步骤进行：

1. **Fork 本仓库**：点击右上角的 Fork 按钮，将本仓库复制到你的 GitHub 账户中。
2. **克隆仓库到本地**：在你的终端中运行以下命令：
    ```sh
    git clone https://github.com/你的用户名/XMU-cs-exam.git
    ```
3. **创建新分支**：在本地仓库中创建一个新的分支：
    ```sh
    git checkout -b feature/你的分支名
    ```
4. **提交你的更改**：将你的作业代码或试卷答案添加到相应的文件夹中，并提交更改：
    ```sh
    git add .
    git commit -m "添加作业代码/试卷答案"
    ```
5. **推送到 GitHub**：将你的更改推送到你的 GitHub 仓库：
    ```sh
    git push origin feature/你的分支名
    ```
6. **创建 Pull Request**：在 GitHub 上提交一个 Pull Request，我们会尽快审核并合并你的更改。
